- [Deno Deploy](https://deno.com/deploy)
- Vanilla TRPCClient in Deno

[Install Deno](https://deno.com/manual/getting_started/installation) then run:

```sh
deno task server
```

Run the client in another terminal:

```sh
deno task client
```
