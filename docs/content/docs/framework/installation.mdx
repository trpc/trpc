## Full-Stack Setup

For complete full-stack applications, you'll need both:

<Steps>
  <Step title="Server Setup">
    Start with the server to define your API:
    
    ```bash
    npm install @trpc/server zod
    ```
    
    [Complete server setup guide →](/docs/server/installation)
  </Step>

  <Step title="Client Setup">
    Then add the client to connect your frontend:
    
    ```bash
    npm install @trpc/client @trpc/react-query @tanstack/react-query
    ```
    
    [Complete client setup guide →](/docs/client/installation)
  </Step>
</Steps>
